Black-Box Modelling of Multiband Saturation — Audio Examples

This project explores black-box modelling of a multiband saturation VST plugin (FabFilter Saturn 2) applied to electric bass guitar. Two architectures were trained and compared on clean and saturated audio pairs:

- Long Short-Term Memory (LSTM): Recurrent Neural Network

- Wavenet: Dilated Causal Convolutional Neural Network
